# Sustentacao

For support work:

1. Load the card or incident context if available.
2. Extract affected customer, environment, identifiers, timestamps, and symptom.
3. Use N1 for triage and known operational checks.
4. Escalate to N2 when root cause requires code, logs, or specialist validation.
5. Preserve evidence, confidence, blockers, and next action.

If providers are unavailable, ask for the relevant card/log/context extract and
continue with `use_user_supplied_context` or `manual_steps`.
