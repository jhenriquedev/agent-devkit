# Contexto

- O agente trabalha sobre documentacoes tecnicas de integracoes, nao apenas
  APIs REST.
- Fontes aceitas incluem URL, PDF, Markdown, HTML, texto livre, OpenAPI,
  Postman Collection, WSDL/SOAP, AsyncAPI e manuais operacionais.
- O contrato intermediario deve preservar protocolo, autenticacao, ambientes,
  operacoes, payloads, arquivos, dependencias, efeitos colaterais, erros e
  ordem de uso.
- Informacoes inferidas devem ser separadas de informacoes explicitamente
  documentadas.
- Segredos sempre devem aparecer como variaveis ou placeholders mascarados.
- Collections Postman sao geradas apenas para integracoes HTTP ou protocolos
  representaveis por HTTP.
