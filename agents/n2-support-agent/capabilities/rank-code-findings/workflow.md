# Rank Code Findings

## Fluxo

1. Carregar contexto.
2. Executar analise de codigo.
3. Ler `technicalFindings`.
4. Ordenar source antes de migration.
5. Ordenar migration antes de test.
6. Ordenar test antes de support.
7. Usar score como desempate.
8. Atribuir rank sequencial.

## Saida

Retorna `rankedFindings` pronto para orientar patch plan.
