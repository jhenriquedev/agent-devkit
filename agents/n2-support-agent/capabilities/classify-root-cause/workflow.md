# Classify Root Cause

## Fluxo

1. Carregar contexto N2.
2. Analisar codigo quando houver `--codebase-path`.
3. Montar texto de evidencia.
4. Detectar sinais de dados, provider externo e backend.
5. Aplicar taxonomia fixa.
6. Atribuir confianca base.
7. Listar evidencias, contradicoes e lacunas.

## Saida

Retorna causa raiz classificada e explicada.
