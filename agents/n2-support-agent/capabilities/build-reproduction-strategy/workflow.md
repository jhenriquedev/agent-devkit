# Build Reproduction Strategy

## Fluxo

1. Carregar contexto N2.
2. Analisar codigo.
3. Escolher primeiro arquivo candidato.
4. Inferir arquivo de teste.
5. Montar `given` com evidencias.
6. Montar `when` com sintoma.
7. Montar `then` com comportamento esperado.
8. Listar passos red, green e refactor.

## Saida

Retorna plano TDD de reproducao.
