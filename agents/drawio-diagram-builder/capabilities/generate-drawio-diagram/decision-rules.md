# Regras

- Gerar arquivo `.drawio` editavel em XML `mxfile` nao comprimido.
- Usar spec intermediaria como fonte da verdade quando fornecida ou gerar spec antes de renderizar.
- Incluir titulo, labels de nodes e conectores com source/target existentes.
- Rotular conectores que representem acao, evento, decisao, protocolo ou payload.
- Incluir legenda quando cor, grupo, tipo de linha ou shape tiver significado.
- Nao sobrescrever arquivo existente sem confirmacao por `yes_overwrite`.
