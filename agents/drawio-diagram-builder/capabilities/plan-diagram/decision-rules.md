# Regras

- Produzir plano com objetivo, audiencia, escopo, fora de escopo, tipo de diagrama e nivel de detalhe.
- Listar fontes rastreadas, elementos obrigatorios, grupos, conectores e perguntas abertas.
- Declarar quality gates esperados: XML parseavel, raiz `mxfile`, labels, conectores validos, geometria, titulo e legenda.
- Recomendar split quando a pagina ficaria poluida ou misturaria familias.
- Definir direcao de leitura: esquerda-direita para fluxos e topo-base para hierarquias.
- Nao criar arquivo `.drawio`; esta capability prepara o plano.
