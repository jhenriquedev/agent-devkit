# Regras

- Usar para entidades, tabelas, chaves, cardinalidade, dominios de dados, linhagem ou ETL.
- Extrair entidades de tabelas, substantivos de dominio ou estruturas explicitas da fonte.
- Inferir relacionamentos por campos `*_id` somente como inferencia, nao como fato confirmado.
- Rotular relacionamentos e cardinalidade quando houver evidencia.
- Dividir schemas grandes por bounded context ou dominio.
- Gerar `.drawio` editavel e validar XML, labels, conectores, geometria e legenda quando houver estilos semanticos.
