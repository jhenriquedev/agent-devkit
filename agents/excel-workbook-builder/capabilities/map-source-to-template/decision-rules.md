# Regras

- Mapear colunas de origem para sheets, ranges e campos declarados do template.
- Nao preencher workbook nesta capability; gerar apenas mapping revisavel.
- Separar correspondencias exatas de correspondencias inferidas.
- Marcar campos obrigatorios sem origem como lacunas.
- Nao alterar template nem dados de origem.
- Exigir revisao humana quando mapeamento for ambiguo ou de alto impacto.
