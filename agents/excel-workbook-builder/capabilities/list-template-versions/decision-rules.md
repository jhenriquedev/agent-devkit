# Regras

- Listar versoes de um template sem alterar metadados.
- Mostrar status, versao atual, depreciadas e caminho rastreavel quando disponivel.
- Tratar template inexistente como erro de catalogo.
- Nao inferir versao atual quando metadados estiverem ausentes.
- Recomendar `inspect-template` antes de usar versao desconhecida.
- Manter saida compacta para uso por outros agentes.
