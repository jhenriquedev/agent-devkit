# Regras

- Gerar arquivo de entrada a partir de template e versao rastreaveis.
- Preservar schema, obrigatoriedade, tipos esperados e validacoes do template.
- Nao modificar o template registrado.
- Incluir instrucoes de preenchimento sem expor regras internas desnecessarias.
- Sinalizar campos ambíguos ou sem descricao.
- Gravar artefato como saida derivada da versao, nao como nova versao do template.
