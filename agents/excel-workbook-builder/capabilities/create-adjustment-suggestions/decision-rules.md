# Regras

- Gerar sugestoes a partir de divergencias classificadas, nunca aplicar ajustes automaticamente.
- Preservar referencia para chave, lado de origem, valor esperado, valor observado e tolerancia.
- Separar sugestao contabil/operacional de fato conciliado.
- Marcar sugestoes de baixa confianca quando faltar regra de negocio.
- Gerar arquivo de saida separado e nao alterar datasets ou workbooks de origem.
- Incluir premissas e duvidas pendentes no artefato.
