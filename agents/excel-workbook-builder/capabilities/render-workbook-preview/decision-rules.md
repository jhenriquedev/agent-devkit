# Regras

- Renderizar abas ou ranges criticos para validacao visual em modo read-only.
- Nao tratar preview como substituto do arquivo `.xlsx`.
- Destacar problemas de layout, cortes, colunas ocultas e areas vazias inesperadas.
- Usar preview como quality gate para workbooks gerados.
- Nao alterar workbook durante renderizacao.
- Registrar aba/range renderizado e limitações do preview.
