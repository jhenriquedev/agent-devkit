# Regras

- Registrar template somente com `yes_save` ou confirmacao equivalente.
- Criar estrutura versionada com template, metadados, input schema e sheet map.
- Nunca sobrescrever template validado existente.
- Validar workbook recebido antes de registrar como reutilizavel.
- Registrar status, versao, nome e rastreabilidade da origem.
- Sinalizar gaps de schema, formulas ou validacoes antes de salvar.
