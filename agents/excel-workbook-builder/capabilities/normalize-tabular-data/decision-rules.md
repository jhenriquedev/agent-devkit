# Regras

- Normalizar nomes de colunas, tipos, datas, numeros e blanks sem alterar a fonte.
- Declarar transformacoes aplicadas e valores que nao puderam ser convertidos.
- Preservar coluna original ou rastreabilidade quando houver conversao ambigua.
- Nao descartar linhas silenciosamente.
- Mascarar dados sensiveis em amostras de erro.
- Gerar JSON normalizado como artefato derivado.
