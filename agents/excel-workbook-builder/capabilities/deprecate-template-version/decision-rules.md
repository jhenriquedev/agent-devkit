# Regras

- Depreciar versao somente com confirmacao explicita.
- Nunca apagar arquivos de template; alterar apenas metadados de status.
- Verificar se existe versao atual ou substituta antes de depreciar.
- Registrar motivo, impacto e data da depreciacao quando possivel.
- Nao depreciar a unica versao valida sem sinalizar risco.
- Preservar historico para auditoria.
