# Regras

- Comparar estrutura, schemas, formulas, validacoes e notas sem alterar templates.
- Identificar diferencas que quebrem compatibilidade com entradas existentes.
- Tratar ausencia de versao como erro de rastreabilidade, nao como versao vazia.
- Destacar mudancas em formulas e ranges nomeados como alto impacto.
- Nao promover, depreciar ou sobrescrever versoes nesta capability.
- Registrar lacunas quando metadados de versao estiverem incompletos.
