# Regras

- Construir dashboard apenas a partir de metricas explicitas.
- Separar KPIs, graficos e callouts de dados brutos.
- Nao inventar metas, thresholds ou status sem regra informada.
- Usar formulas auditaveis para indicadores derivados.
- Preservar abas existentes e gerar copia de saida.
- Exigir `render-workbook-preview` e `review-generated-workbook` antes de considerar entrega final.
