# Regras

- Gerar `.xlsx` a partir de dados normalizados e tipados.
- Preservar dados de entrada e gravar workbook em arquivo separado.
- Criar abas claras para dados, resumo e metadados quando aplicavel.
- Evitar formulas hardcoded; usar formulas auditaveis para calculos derivados.
- Registrar fonte, schema e premissas usadas.
- Recomendar preview, scan de formulas e review antes da entrega.
