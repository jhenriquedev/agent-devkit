# Regras

- Atualizar somente a aba de dados alvo, preservando formulas, outras abas e layout.
- Criar arquivo de saida separado; nao sobrescrever workbook original silenciosamente.
- Validar schema da nova entrada antes de substituir dados.
- Reportar linhas adicionadas, removidas e alteradas quando possivel.
- Executar ou recomendar review e scan de formulas apos refresh.
- Nao alterar template ou formulas fora da area de dados.
