# Regras de Formula

- Use formulas para calculos derivados.
- Evite numeros magicos dentro de formulas.
- Use abas de parametros para premissas reutilizadas.
- Referencie abas com aspas simples quando houver espacos ou caracteres
  especiais: `='Dados Brutos'!A1`.
- Escaneie erros como `#REF!`, `#DIV/0!`, `#VALUE!`, `#NAME?` e `#N/A`.
