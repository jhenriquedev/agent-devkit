# Regras de Conciliacao

- A chave de conciliacao deve ser explicita.
- Diferencas devem ser classificadas como `matched`, `different`,
  `left_only` ou `right_only`.
- Tolerancia numerica deve ser registrada quando usada.
- O workbook de conciliacao deve conter resumo e detalhes auditaveis.
- Sugestoes de ajuste nunca devem ser aplicadas automaticamente.
