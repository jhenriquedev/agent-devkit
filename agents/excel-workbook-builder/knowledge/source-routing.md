# Roteamento de Fontes

- CSV/TSV/JSON: processar localmente.
- XLSX simples: importar/inspecionar quando necessario.
- PDF/DOCX: extrair texto/tabelas quando a estrutura permitir.
- SQL Server: delegar para `sqlserver-data-analyzer`.
- PostgreSQL: delegar para `postgres-data-analyzer`.
- Azure DevOps: delegar para `azure-devops-orchestrator`.
