# Workflow: Generate Knowledge

1. Inspecionar a fonte.
2. Resolver profile informado ou recomendar automaticamente.
3. Criar a pasta de saida quando `--yes-create-dir` for informado.
4. Gerar artefatos obrigatorios do profile.
5. Registrar lacunas e arquivos ignorados.
6. Validar JSON e presenca de artefatos minimos.
