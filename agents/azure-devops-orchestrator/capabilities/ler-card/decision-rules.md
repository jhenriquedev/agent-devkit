# Decision Rules: Ler Card

- Sempre separar fatos coletados de inferencias.
- Se comentarios forem solicitados, resumir apenas os relevantes para decisao.
- Se houver criterio de aceite ausente, marcar como lacuna.
- Se houver conflito entre descricao e comentarios, destacar o conflito.
- Nao recomendar escrita como se ela ja tivesse sido executada.
