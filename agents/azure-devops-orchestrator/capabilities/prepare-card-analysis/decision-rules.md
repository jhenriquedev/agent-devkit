# Decision Rules: Preparar Analise Card

- Separar fatos de hipoteses.
- Usar linguagem de confianca moderada quando faltar evidencia.
- Nao gerar comentario como se tivesse sido publicado.
- Se criterio de aceite ou responsavel estiverem ausentes, registrar lacuna.
- Para publicar comentario, orientar uso de `comment-card`.
