# Decision Rules: Listar Cards

- Se o usuario informar query, usar query antes de filtros inferidos.
- Se nao houver limite, usar limite seguro definido pelo repository.
- Se o filtro retornar muitos cards, pedir refinamento antes de aprofundar.
- Nao priorizar cards automaticamente sem criterio explicito.
- Nao executar qualquer escrita nesta capability.
