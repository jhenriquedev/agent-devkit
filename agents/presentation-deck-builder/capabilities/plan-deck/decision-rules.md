# Decision Rules

- Planejar slides a partir de conteudo extraido e restricoes do template.
- Verificar campos obrigatorios antes de afirmar que o plano esta pronto.
- Perguntar quando conteudo estiver incompleto, contraditorio ou ambiguo.
- Nao inventar fatos, metricas, conclusoes ou narrativa que nao esteja no input.
- Mapear cada bloco de conteudo para slide, layout e placeholder esperado.
- Evitar slides superlotados; dividir conteudo quando o template exigir.
- Preservar ordem narrativa adequada ao publico e objetivo informados.
- Registrar lacunas que bloqueiam `generate-deck-from-template`.
- Nao gerar PPTX nesta capability.
- A saida deve ser diretamente validavel contra o input schema.
