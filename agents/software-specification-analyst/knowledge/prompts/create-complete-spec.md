# Prompt: Create Complete Spec

Gere uma especificacao completa de software em portugues, usando a demanda de
entrada como fonte. Separe fatos, inferencias, premissas e perguntas abertas.

Antes de concluir, aplique os quality gates de `knowledge/policies.yaml`.

Use Mermaid para jornadas e fluxos. Nao invente requisitos: quando faltar
informacao, registre em "Perguntas Abertas".
