# Workflow: Create Functional Spec

1. Identificar problema, objetivo, atores e superficies.
2. Separar escopo, fora de escopo e regras de negocio.
3. Numerar requisitos funcionais.
4. Mapear fluxos principais, alternativos e excecoes.
5. Registrar premissas e perguntas abertas.
6. Validar criterios de aceite por requisito.
