# Workflow: Create Complete Spec

1. Ler a demanda de entrada.
2. Consultar `vendor/skills/CATALOG.md` e selecionar apenas skills relevantes.
3. Usar `product-capability` como base para explicitar contrato, restricoes e
   perguntas abertas.
4. Separar fatos, inferencias, premissas e lacunas.
5. Gerar especificacao completa em Markdown.
6. Propor a pasta `<projeto-atual>/specifications/<slug-da-demanda>/`.
7. Perguntar antes de criar a pasta quando ela ainda nao existir.
8. Perguntar antes de sobrescrever arquivos existentes.
9. Salvar os artefatos aprovados.
10. Encerrar com lista de arquivos gerados e pendencias relevantes.
