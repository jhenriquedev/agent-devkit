# Regras

- Avaliar baseline com accuracy, precision, recall, F1 e matriz de confusao.
- Declarar classe positiva, threshold ou regra usada.
- Tratar metricas como avaliacao do baseline implementado, nao de modelo externo.
- Sinalizar desbalanceamento, leakage e amostra pequena.
- Separar metricas de recomendacoes operacionais.
- Mascarar dados pessoais em exemplos de erro.
