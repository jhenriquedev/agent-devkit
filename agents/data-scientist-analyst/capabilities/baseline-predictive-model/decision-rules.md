# Regras

- Treinar apenas baseline simples e explicavel por threshold numerico.
- Declarar feature escolhida, regra, threshold e metricas in-sample.
- Nao apresentar metricas in-sample como performance de producao.

- Reportar desbalanceamento de classe e baixa contagem como limitacoes.
- Nao persistir modelo nem alterar fonte.
