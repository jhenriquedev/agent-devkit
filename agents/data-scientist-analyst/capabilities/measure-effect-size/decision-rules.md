# Regras

- Calcular Cohen's d apenas para dois grupos e metrica numerica.
- Declarar medias, desvios, tamanhos de grupo e magnitude pratica.
- Interpretar tamanho de efeito separado de significancia estatistica.
- Sinalizar grupos pequenos, variancias muito diferentes ou outliers fortes.
- Nao afirmar impacto causal sem desenho experimental apropriado.
- Mascarar valores sensiveis em qualquer amostra de grupo.
