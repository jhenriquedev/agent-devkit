# Regras

- Comparar referencia e dataset atual por deslocamento de media normalizado.
- Declarar colunas analisadas, medias, desvios e limiares usados.
- Tratar drift como alerta estatistico, nao como falha de modelo confirmada.
- Sinalizar mudancas de schema, nulos e baixa variancia.
- Nao alterar dados nem recalibrar modelo nesta capability.
- Mascarar dados pessoais em exemplos de distribuicao.
