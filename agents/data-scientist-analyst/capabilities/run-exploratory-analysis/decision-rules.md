# Regras

- Separar estatisticas observadas, hipoteses exploratorias e limitacoes.
- Nao apresentar correlacao, segmentacao ou outlier como causalidade.
- Mascarar dados pessoais em exemplos e segmentos de baixa contagem.
- Declarar colunas usadas como alvo, segmento e metricas.
- Reportar lacunas quando dados estiverem incompletos, desbalanceados ou amostra for pequena.
- Gerar artefato somente quando `output` for informado.
