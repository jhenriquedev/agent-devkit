# Regras

- Calcular intervalo para media numerica usando aproximacao normal.
- Declarar nivel de confianca, media, erro padrao, n e limites.
- Sinalizar baixa amostra, nulos ou distribuicao assimetrica como limitacao.
- Nao interpretar intervalo como probabilidade da media verdadeira estar dentro apos calculado.
- Nao aplicar a colunas nao numericas.
- Registrar fonte e coluna analisada.
