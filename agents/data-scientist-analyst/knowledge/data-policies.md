# Politicas De Dados

- Fontes sao somente leitura.
- Amostras e relatorios mascaram dados pessoais por padrao.
- Arquivos gerados devem declarar fonte, hash e regras usadas.
- Divergencias de conciliacao devem ser explicadas com motivo objetivo.
- Dados sensiveis nao devem ser expostos integralmente em stdout.
- Mutacoes em bancos, planilhas ou sistemas externos sao fora de escopo.
