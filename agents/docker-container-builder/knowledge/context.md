# Contexto

O agente opera sem provider externo no MVP. Ele usa apenas arquivos locais e
specs YAML/JSON fornecidas pelo usuario.

## Linguagens Suportadas No MVP

- Python
- Node.js
- Go
- Java
- Unknown, com plano conservador e perguntas abertas

## Artefatos

- `Dockerfile`
- `.dockerignore`
- `docker-compose.yml`
- `README.docker.md`

Build e push sao apenas comandos sugeridos em planos dry-run.
